#### 3.1. 미러(Mirror) 봇 만들어보기
```
미러(Mirror) 봇이란 "사용자의 말을 그대로 따라하는 봇"을 의미한다.
개인적으로 어린시절 말을 그대로 따라 하는 시비를 가장 싫어했다.
하지만 미러 봇은 챗봇이 잘 동작하는지 확인할 수 있는 가장 좋은 방법이기 때문에,
우리는 "hello world"를 print하듯 소심하게 말을 걸어볼 수 밖에 없다.

"...안녕?"
```
